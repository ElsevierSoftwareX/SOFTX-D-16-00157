Time-frequency reassignment (functions)

1. tf_image* reassign_time_frequency(tf_image* a_sdelay, tf_image* a_ifrequency, tf_image* a_senergy, tf_image* a_output)

This function realizes simple time-frequency reassignment. The output image can have different time-frequency configuration as well as width and height. These informations are taken from the last argument (a_output) - have to be allocated. If last argument is NULL then memory is allocated for output object independently.

Arguments and return:
  • a_sdelay - spectral delay,
  • a_ifrequency - instantaneous frequency,
  • a_senergy - spectral energy,
  • a_output - object to which energy is added (default value is NULL),
  • return - reassigned energy.

    Frequency reassignment (functions)

    1. tf_image* reassign_frequency(tf_image* a_ifrequency, tf_image* a_senergy, tf_image* a_output)
    2. tf_image* reassign_frequency(tf_image* a_sdelay, tf_image* a_crate, tf_image* a_ifrequency, tf_image* a_senergy, tf_image* a_output)

    These functions realize the reassignment only in frequency. The output image can have different time-frequency configuration as well as width and height. These informations are taken from the last argument (a_output) - output have to be before allocated. Otherwise, if last argument is equal to NULL, then memory is allocated for output object independently.

    Arguments and return:
  • a_sdelay - spectral delay,
  • a_crate - chirp rate,
  • a_ifrequency - instantaneous frequency,
  • a_senergy - spectral energy,
  • a_output - object to which energy is added (default value is NULL),
  • return - reassigned energy.

    Energy profile (histogram) (function)

    1. tfr_complex_signal* calculate_profile(tf_image* a_val, tf_image* a_energy, tfr_array_config* a_arr_conf)

    This function calculates energy profile and histogram as a function of a given parameter (as the TF distribution) and it returns its as tfr_complex_signal. The energy profile is its real part, the histogram is saved in its imaginary part.

    Arguments and return:
  • a_val - TF distribution of a parameter,
  • a_energy - spectral energy in TF domain,
  • a_arr_conf - configuration of profile (histogram),
  • return - signal storing the profile and histogram.
