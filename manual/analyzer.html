Analyzer construction

1. tfr_iir_analyzer(tfr_array_config* a_bank_conf, tfr_filter_config* a_filter_conf)
2. tfr_fft_analyzer(tfr_array_config* a_bank_conf, tfr_window_config* a_window_conf)

The first constructor creates an object using the structure of bank configuration for set center frequencies of filters. The second structure defines other filter parameters. The secont is yet not implemented.

Arguments and return:
  • a_filter_conf - pointer to filter configuration,
  • a_window_conf - pointer to window configuration,
  • a_bank_conf - pointer to bank configuration.
    •
    Set signal (method)

    1. void set_signal(tfr_complex_signal* a_signal, int a_hop)

    The function allows to set analyzed signal. Filtering is made here automatically.

    Arguments and return:
  • a_signal - input signal,
  • a_hop - decimation factor (default value is 1).
