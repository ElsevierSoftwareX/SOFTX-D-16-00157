    Get configuration (methods)

    1. tfr_array_config* get_array_configuration()
    2. tfr_array_config* get_config()

    These functions return object configuration as a pointer to a tfr_array_config object. The configuration concens frequency axis. Both functions work identically.

    Arguments and return:
  • return - frequency axis configuration.

    Get rate (methods)

    1. double get_sampling_rate()
    2. double get_rate()

    These functions return acceptable sampling rate. This value is set when a first signal is analyzed. Both functions work identically.

    Arguments and return:
  • return - acceptable sampling rate.
    •
    Add signal (methods)

    1. double add_complex_signal(tfr_complex_signal* a_signal, int a_hop)
    2. double add_signal(tfr_complex_signal* a_signal, int a_hop)

    These functions make segmentation, transform, and estimation of spectral energy density for each segmented frame. Then, these distributions are added to the periodogram. The functions return amount of added energy. Both functions work identically.

    Arguments and return:
  • a_signal - analyzed signal.
  • a_hop - hopsize between consecutive frames.
  • return - amount of added energy.
    •
    Energy calculation (method)

    1. double calc_energy()

    This function can be used for energy calculation of the periodogram.

    Arguments and return:
  • return - amount of periodogram energy.
