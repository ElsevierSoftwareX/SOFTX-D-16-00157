tfr_pulse_generator

This class introduces generator definition. The generator is dedicated to produce periodic frequency modulated pulse signals (pulse series). There are available three types of pulses: harmonic, linear frequency modulated, as well as hyperbolic frequency modulated. The pulses can be shaped (tapered) by Tukey window. The transitional range of this window can be specified by 'transition' field in tfr_pulse_config structure.

    Add signal (methods)

    1. double add_harmonic(tfr_complex_signal* a_sig)
    2. double add_lfm_chirp(tfr_complex_signal* a_sig)
    3. double add_hfm_chirp(tfr_complex_signal* a_sig)

    These functions allow to add to existing signal new generated samples. The generator can produce shaped pulses of harmonic signals, linear frequency modulated chirps, as well as hiperbolic modulated chirps. A new signal is generated based on internal generator configuration. Each pulse is tapered by Tukey window. The field transition in generator_config is used for set range of tapered margins (value between 0 and 1).

    Returned value:
  • a_sig - existing signal, which will be modified,
  • return - added energy.
    •
    Add random (method)

    1. double add_random_fsk(tfr_complex_signal* a_sig)

    This function allows to add to existing signal new generated samples. The generator can produce shaped FSK pulses whose frequency is random. Each pulse is tapered by Tukey window similar as in the previous section (Add signal).

    Returned value:
  • a_sig - existing signal, which will be modified,
  • return - added energy.
