Filter construction

1. tfr_recursive_filter(tfr_filter_config* a_conf)

We provide the constructor which uses the filter structure.

Argument:
  • a_conf - filter configuration.
    •
    Get configuration (methods)

    1. tfr_filter_config* get_filter_configuration()
    2. tfr_filter_config* get_config()

    These functions return object configuration as a pointer to a tfr_filter_config object. Both functions work identically.

    Arguments and return:
  • return - filter configuration.
    •
    Filtering (methods)

    1. complex double filtering(complex double a_sample)
    2. tfr_complex_signal* filtering(tfr_complex_signal* a_signal, int a_hop)

    These functions realize filtering operation. The first function gets single input sample and return single output sample. The second works with the whole input signal, then output signal is allocated and its pointer is returned.

    Arguments and return:
  • a_sample - single input complex-valued sample,
  • a_signal - input signal,
  • a_hop - decimation factor (default value is 1),
  • return (first) - output sample,
  • return (second) - output signal.
    •
    P factor (method)

    1. complex double get_p_factor()

    The function returns the complex p factor.

    Arguments and return:
  • return - p factor.

    1. int get_processed_sample_number()
