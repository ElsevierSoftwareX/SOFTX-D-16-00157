Image construction

1. tfr_raster_image(tfr_image_config* a_conf)

The constructor creates an empty object. The memory is allocated, however sample values in internal buffers are equal to zeros.

Arguments and return:
  • a_conf - image configuration.
    •
    Get configuration (method)

    1. tfr_image_config* get_image_configuration()
    2. tfr_image_config* get_config()

    These functions return object configuration. Both functions work identically.

    Arguments and return:
  • return - structure with image configuration.
    •
    Compare configuration (method)

    1. bool compare_configuration(tfr_image_config* a_conf)

    This function returns true if an object and its argument (object which is pointed by argument) have the same time-frequency configurations, widths, and heights.

    Arguments and return:
  • a_conf - pointer to tfr_image_config which is compared,
  • return - binary result of comparison.
    •
    Get index (method)

    1. int get_index_by_time(double a_time)
    2. int get_index_by_frequency(double a_frequency)

    These functions return an index number which is the nearest in time (or in frequency) to the argument with respect to the time-frequency configuration.

    Arguments and return:
  • a_time - checked time,
  • a_frequency - checked frequency,
  • return - index number.
    •
    Get time / frequency (methods)

    1. double get_time_by_index(int a_time_index)
    2. double get_frequency_by_index(int a_frequenc_indexy)

    These functions return time or frequency which corresponds to a given index.

    Arguments and return:
  • a_time_index - time index,
  • a_frequency_index - frequency index,
  • return (first)- corresponding time,
  • return (second)- corresponding frequency.
    •
    Get marginal distributions (methods)

    1. tfr_real_array* get_horizontal_distribution(tfr_raster_image* a_energy)
    2. tfr_real_array* get_vertical_distribution(tfr_raster_image* a_energy)

    These functions return marginal distributions (also weighted).

    Arguments and return:
  • a_energy - energy distribution which is used as weight (default value is NULL),
  • return (first) - horizontal marginal distribution,
  • return (second) - vertical marginal distribution.
    •
    Save (methods)

    1. void save_to_file(const char* a_filename)
    2. void save(const char* a_filename)

    These functions saves data to a text file. Both functions work identically.

    Arguments and return:
  • a_filename - file name which will be created.
    •
    Clear (methods)

    1. void clear_data()
    2. void clear()

    These functions clear data, however do not free memory. Both functions work identically.
