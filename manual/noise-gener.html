White noise adding (methods)

1. double add_awgn_using_snr(tfr_complex_signal* a_sig, double a_snr)
2. double add_awgn_using_sigma(tfr_complex_signal* a_sig, double a_sigma)
3. double add_awgn_at_exact_snr(tfr_complex_signal* a_sig, double a_snr)

These functions add pseudo white noise to the waveform of the signal object which is given as an argument. First mehod uses signal-to-noise ratio (SNR), second uses sigma, and thrid also SNR. It is The thrid is slower than first. However, it adds noise with exact specified SNR. These functions add noise to the whole signal.

Arguments and return:
  • a_snr - signal-to-noise ratio,
  • a_sigma - sigma used for noise generation,
  • a_sig - existing signal, which will be modified,
  • return - added energy.

    2. double add_pink_at_exact_snr(tfr_complex_signal* a_sig, double a_snr)

    These functions add pseudo-pink noise (1/f) to the waveform of the signal object which is given as an argument. We are based on the description from this webpage. First mehod uses range, second uses signal-to-noise ratio. These functions add noise to the whole signal.

    Arguments and return:
  • a_snr - signal-to-noise ratio,
