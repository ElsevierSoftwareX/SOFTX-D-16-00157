Window generator construction

1. tfr_window_generator(tfr_window_config* a_window_conf)

The constructor creates an object using the structure of window generator structure.

Arguments and return:
  • a_window_conf - pointer to window configuration.

    Get window (methods)

    1. tfr_complex_signal* get_window(int a_d_order, int a_t_order)
    2. tfr_complex_signal* get_window_waveform(int a_d_order, int a_t_order)

    These functions return window waveform as complex_sugnal object. These functions work identically.

    Arguments and return:
  • a_d_order - order of derivative with respect to time (default value is 0),
  • a_t_order - power which is used for multiplication by time ramp (default value is 0),
  • return - pointer to generated window.
    •

    2. tfr_window_config* get_config()

    Arguments and return:
  • return- window configuration.
